## 3월 27일 월요일 학습 계획

> 미션 순서에 따라 구현을 진행하면서 필요한 부분을 찾아 학습한다.

1. 회원 가입 기능 구현 (1H)  
    회원 가입 페이지에서 POST 요청을 보내면 회원 저장 후 회원 목록 조회 페이지로 이동한다.
2. 회원 가입 기능을 구현하면서 알게된 점 정리 (1H)
3. 회원 목록 조회 기능 구현 (1H)  
    회원 목록 페이지에서 GET 요청을 보내 정보를 받아오고 브라우저에 회원 목록을 출력한다.
4. 회원 목록 조회 기능을 구현하면서 알게된 점 정리 (1H)
5. 알고리즘 문제 풀기 (1H)
6. CS 공부하기 (1H)
7. Spring 강의 듣기 (2H)
