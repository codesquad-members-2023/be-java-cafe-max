## 스프링 카페 1단계

### 기능에 따른 URL 컨벤션

| URL                 | 기능           |
|---------------------|--------------|
| POST /user/add      | 회원 가입 기능     |
| GET /user/list      | 회원 목록 조회 기능  |
| GET /user/:userId   | 회원 정보 조회 기능  |
| PUT /user/:userId   | 회원 정보 수정 기능  |
| POST /board/write   | 글 쓰기 기능      |
| GET /board/list     | 글 목록 조회 기능   |
| GET /board/:boardId | 글 상세정보 조회 기능 |
